# Wealthwise Frontend

Simple React TypeScript frontend for the Wealthwise expense management platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run on http://localhost:5173

## Features

- User Authentication (Login/Signup)
- Dashboard with expense overview
- Expense Management (Add, View, Delete)
- Budget Management
- Financial Goals Tracking

## Tech Stack

- React 18
- TypeScript
- React Router
- Vite
- No external UI libraries - pure CSS-in-JS for simplicity
